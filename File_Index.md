# 文件索引

## CSS 文件

| 文件路径 | 作用 | 说明 |
|---------|------|------|
| css/theme.css | 主题样式 | 日间/夜间 CSS 变量定义 |
| css/login.css | 登录样式 | 登录/注册页布局和组件 |
| css/report.css | 报告样式 | 报告展示组件样式 |
| css/request-capsule.css | 胶囊样式 | 请求提示浮动样式 |

## JavaScript 文件

| 文件路径 | 作用 | 说明 |
|---------|------|------|
| js/api.js | 网络模块 | Axios 封装，统一请求管理 |
| js/request.js | 请求类 | Request 类封装 |
| js/theme-manager.js | 主题模块 | 日间/夜间切换逻辑 |
| js/chart-factory.js | 图表工厂 | ECharts 配置生成 |
| js/chart-parser.js | 图表解析 | 数据解析为图表格式 |
| js/report-renderer.js | 报告渲染 | 报告内容渲染引擎 |
| js/blog-editor.js | 编辑器 | 报告编辑 UI 逻辑 |

## HTML 文件

| 文件路径 | 作用 | 说明 |
|---------|------|------|
| html/login.html | 登录页 | 用户登录表单 |
| html/register.html | 注册页 | 用户注册表单 |
| html/show-report.html | 报告展示 | 普通用户查看报告 |
| html/manage.html | 管理中心 | 管理员管理报告 |
| html/blog-editor.html | 编辑器 | 管理员编辑报告 |
| html/simple.html | 示例页 | 主题切换示例 |
| html/report-demo.html | 报告 Demo | 季度报告示例 |
| html/family-impact-chart.html | 图表页 | 家庭影响分析图表 |
| html/api-test.html | API 测试 | 接口测试控制台 |
| html/db-test.html | 数据库测试 | 数据库 JSON 测试 |

## 配置文件

| 文件路径 | 作用 | 说明 |
|---------|------|------|
| WEB-INF/web.xml | Servlet 配置 | 过滤器和 Servlet 映射 |
| package.json | npm 配置 | 项目依赖和脚本 |
